## 1. 行前須知：天氣與梅雨季

### 是的，5 月底～6 月初正是梅雨季（Mei-yu Season）

| 項目 | 內容 |
|------|------|
| **天氣特徵** | 滯留鋒面徘徊，全台容易出現連續陰雨 |
| **氣溫** | 白天 30～33°C，夜間 23～26°C，悶熱潮濕 |
| **濕度** | 80～90%，體感溫度偏高 |
| **降雨型態** | 午後雷陣雨 + 鋒面帶來的持續降雨 |
| **颱風風險** | **低～中等**，主要颱風季為 7～9 月，但仍有形成可能 |

### 必備物品
- ☂️ **摺疊傘 + 輕便雨衣**（隨身攜帶，說下就下）
- 👟 **防水鞋／涼拖鞋**（路面易積水）
- 🧴 **防蚊液**（梅雨季蚊蟲多）
- 🧥 **快乾薄外套**（防曬兼防雨）
- 📱 **下載中央氣象署 App**（CWA，及時看雷達回波）

### 行程調整建議
- 每天上午盡量排戶外行程（通常下午降雨機率更高）
- 每個城市**準備 1～2 個室內備案**（博物館、百貨、溫泉）
- 山區行程（阿里山、陽明山）遇大雨建議延期

## 2. 8 天建議去幾個城市？

### 最佳方案：3～4 個城市（西岸縱走）

對於 8 天來說，**3 個城市最舒適，4 個城市可行但較趕**。

以下為三種主流路線的比較：

| 方案 | 城市數 | 適合誰 | 節奏 |
|------|--------|--------|------|
| **A. 經典西岸縱走** | 4 城（台北→台中→台南→高雄） | 第一次來台灣、想一次看精華 | 中快 |
| **B. 台北＋台南慢遊** | 2 城（台北 5d + 台南 3d） | 深度美食文化之旅 | 悠閒 |
| **C. 台北＋花蓮大自然** | 2～3 城（台北 4d + 花蓮 3d + 台北/高雄 1d） | 愛爬山、自然風景 | 中等 |

> ⭐ **本文件以方案 A「經典西岸縱走」為主軸**，因為 8 天時間最能平衡廣度與深度。

## 3. 推薦路線方案

### 方案 A：經典西岸縱走（最推薦給第一次訪台者）

```
台北 3.5 天 → 高鐵 → 台中 1.5 天 → 高鐵 → 台南 1.5 天 → 台鐵/高鐵 → 高雄 1.5 天
```

**日數分配：**
- **台北：3～4 天**（含九份、十分一日遊）
- **台中：1～2 天**（中繼站，可加日月潭一日遊）
- **台南：1～2 天**（台灣美食之都）
- **高雄：1～2 天**（港都風情）

### 方案 B：台北＋台南深度慢遊（適合不愛趕路的人）

```
台北 5 天 → 高鐵 2 小時 → 台南 3 天
```

適合：美食愛好者、帶長輩、想要真正 relax 的人

### 方案 C：東岸自然風光（愛山愛海的人）

```
台北 3 天 → 台鐵 2 小時 → 花蓮 3 天 → 台鐵/飛機 → 高雄/台北 2 天
```

**注意：** 花蓮到高雄搭台鐵約 4～5 小時，會吃掉半天交通時間。

|------|-------------------|-------------------|----------|

| 台北 → 台南 | 1 時 10 分～1 時 40 分 | **$1,120** | 同上 |
| 台北 → 高雄（左營） | 1 時 34 分～2 時 | **$1,490** | 同上 |
| 台中 → 台南 | 38～70 分 | **$590** | 同上 |
| 台中 → 高雄（左營） | 50 分～1 時 10 分 | **$790** | 同上 |
| 台南 → 高雄（左營） | 14～35 分 | **$370** | 同上 |

- 官網：https://www.thsrc.com.tw
- 建議用「T-EX 行動購票」App 直接用手機訂票、進站刷 QR Code

### 台鐵（TRA）— 省錢替代方案

| 路線 | 車種 | 時間 | 票價 NTD | 備註 |
|------|------|------|----------|------|
| 台北 → 台中 | 自強號 | 約 2 時 10 分 | $381 | 比高鐵慢但便宜近一半 |
| 台南 → 高雄 | 區間車/自強號 | 25～40 分 | **$48～$89** | 大推！只比高鐵慢 10 分，價格 1/4 |

### 市區交通
- **台北：** 捷運（MRT）+ YouBike 共享單車，幾乎不用叫車
- **台中：** 公車（10 公里內免費）+ 台灣大車隊計程車
- **台南：** 公車較不便，建議租機車或使用 Uber/台灣大車隊
- **高雄：** 捷運紅橘線 + 輕軌 + YouBike

## 7. 行程總表 — 完整 8 天建議

### 📋 方案 A 實際行程（西岸縱走）

```
Day 1  — 台北抵達 + 市區探索
Day 2  — 台北：九份、十分一日遊
Day 3  — 台北：故宮 + 北投 + 士林夜市
Day 4  — 上午台北 → 高鐵 → 台中，下午台中市區
Day 5  — 台中：日月潭一日遊
Day 6  — 上午台中 → 高鐵 → 台南，下午台南美食之旅
Day 7  — 台南：安平古堡 + 赤崁樓 + 午餐 → 台鐵 → 高雄，傍晚旗津
Day 8  — 高雄：駁二 + 西子灣 + 佛光山 → 從高雄小港機場返程
```
